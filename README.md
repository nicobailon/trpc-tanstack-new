# trpc with TanStack Start

This example uses `@tanstack/react-start` (updated from the deprecated `@tanstack/start` package).

### Playing around

```bash
pnpm install
pnpm dev
```

> [!TIP]
> Try editing the app.config.ts deployment presets for different runtime builds. :)

## Building

```bash
pnpm build
pnpm start
```
